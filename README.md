> Luminos 源自拉丁语词根 *lumin-*（光），意为「星辉计划」——

```
零安装  ·  零服务器  ·  零依赖

```

- 🎨 **深海星空主题**：精心调配的蓝色渐变 + 动态星点背景，久看不疲

## 📋 功能一览

### 🗓 三层计划体系

```
今日计划  ←→  本周计划  ←→  学期课程
   ↑               ↓
实时计时      截止日期自动推算
```

三层计划**双向联动**：本周任务设置截止日期后，系统自动将需要今天完成的任务推送到「今日安排」区块，提前完成还有额外积分奖励。

-----

### ⏱ 今日计划 · 智能排程

**核心功能：**

|功能      |说明                             |
|--------|-------------------------------|
|📌 固定时间  |指定任务开始时间（如 `08:00`），排程引擎自动绕行   |
|🔄 智能排程  |从当前时刻起自动排列，每 60 分钟插入 10/20 分钟休息|
|⏸ 实时计时  |精确到秒的计时器，超时自动变红提醒，同一时刻只允许一个任务计时|
|📊 可视化时间轴|色块任务 + 餐饮/睡眠背景层 + 实时光标，一目了然    |

**作息时间自定义（可折叠设置面板）：**

```
起床时间  ──────────────────────────────  睡觉时间
           早餐 │    午餐    │  晚餐
           ↑              ↑          ↑
        排程引擎自动跳过这些时段
```

- 自由设置起床/睡觉时间
- 默认三餐可修改开始时间和时长
- 可添加任意数量额外用餐时段（下午茶、夜宵等）

-----

### 📅 本周计划 · 截止日期智能推算

```
添加任务 → 填写截止日期
              ↓
    系统推算建议完成日（截止前 1 天）
              ↓
    今日需完成的推送至「今日安排」
              ↓
    提前完成 → 额外 +8 积分 🎉
```

- 七列周视图，高/中/低优先级色条标识
- 每项任务显示截止日期和预计用时
- 今日建议完成的任务青蓝高亮
- 支持编辑：名称、用时、截止日期、优先级

### 📚 学期课程 · 章节掌握追踪

**这是 Luminos 最独特的功能。**

不同于简单的任务打卡，Luminos 以**科目为单位**追踪学习深度：

```
科目卡片
├── 进度条（已掌握章节 / 总章节）
├── 标签：✓掌握N · ~学过N · ✗未学N
└── 可展开章节列表
       ├── Ch.1  第一章  ⭐⭐⭐⭐⭐  (大师)
       ├── Ch.2  第二章  ⭐⭐⭐      (掌握)
       ├── Ch.3  第三章  ⭐          (了解)
       └── Ch.4  第四章  ···         (未学)
```

**五星自评体系：**

|星级   |掌握程度|积分奖励       |
|-----|----|-----------|
|⭐    |了解  |+1 pts     |
|⭐⭐   |熟悉  |+3 pts（累计） |
|⭐⭐⭐  |掌握  |+5 pts（累计） |
|⭐⭐⭐⭐ |精通  |+8 pts（累计） |
|⭐⭐⭐⭐⭐|大师  |+12 pts（累计）|


> 积分按**差值**结算：从「了解」升到「掌握」获得 +4pts（5-1=4）；降级同样扣回——杜绝刷分。

**学期进度条 + 里程碑：**

```
W1 ──── W4 ──── W8 ────[📐]── W12 ────[📘]── W16 ──── W18
                       ↑ 线代考试           ↑ 物理考试
```

- 设置学期起始日期后，**自动推算当前周次**
- 考试日期换算为周次，在进度条上显示里程碑
- 临近考试时进度条里程碑发出青蓝色光晕

### ✦ 积分成长体系

#### 积分获取规则

|行为             |积分      |
|---------------|--------|
|✅ 完成今日任务       |+10     |
|⏱ 按时完成（不超时 10%）|+5      |
|🔥 连续完成 3 项     |+15     |
|📅 提前完成周任务      |+8      |
|🎯 今日全部完成       |+20     |
|🎓 完成科目（考试）     |+20     |
|⭐ 章节掌握（按等级）    |+1 ~ +12|

#### 六级成长称号

```
🌑 星尘旅者  →  🌙 银河探险者  →  ✨ 星云研究员
                                        ↓
🌌 宇宙缔造者  ←  💫 星际指挥官  ←  🌟 宇宙学者
```

#### 自定义兑换商店

默认提供 8 种兑换项（奶茶、游戏、追剧…），完全支持自定义：

- **添加**：自定义名称 + Emoji 图标 + 所需积分
- **编辑**：随时调整任意字段
- **删除**：移除不需要的奖励
- **恢复默认**：一键重置为默认列表

> 💡 建议根据自己的实际奖励定制商店，学习驱动力因人而异。

## 🔐 账户与数据

```
注册 / 登录
   ↓
localStorage['lp_d_用户名']  ← 用户数据
localStorage['lp_u']         ← 账户列表（密码哈希存储）
localStorage['lp_s']         ← 当前会话（自动登录）
localStorage['lp_lb']        ← 共享排行榜
```

- 密码经过哈希处理，不明文存储
- 每个账户数据完全隔离
- 每日自动重置今日任务、今日积分和连击数
- 每 30 秒自动保存，操作触发立即保存

> ⚠️ 数据存储在浏览器本地，清除浏览器数据会丢失。如需跨设备同步，需自行搭建后端（见下文）。

## 🛠 技术栈

```
纯原生前端，零框架，零构建工具

HTML5        · 语义化结构
CSS3         · 变量系统 / Flexbox / Grid / 动画
Vanilla JS   · ES2020+ / 模块化函数设计
localStorage · 持久化数据存储
```

**架构设计：**

```
State (S)  ──→  Render Functions  ──→  DOM
   ↑                                    │
   └──────────  Event Handlers  ←───────┘
                     │
               saveAll() → localStorage
```

单一状态对象 `S` 驱动所有渲染，类 Flux 的单向数据流，便于扩展。
